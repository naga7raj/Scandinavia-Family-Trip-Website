# Scandinavia Family Trip — Final Family Version

A polished, mobile-friendly family travel dashboard for the July 2026 journey through Bergen, Oslo and Copenhagen.

## Features

- Detailed day-by-day itinerary
- Google Maps shortcuts
- Booking tracker
- Packing checklist saved in the browser
- Dark mode
- Countdown to departure
- Budget, travel apps and emergency information
- Mobile responsive design

## Privacy

This public version intentionally excludes booking PINs, security codes, QR codes, payment details, personal documents and exact accommodation addresses.
