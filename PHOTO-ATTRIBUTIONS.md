# Photo Attributions

The website uses real destination photographs from Wikimedia Commons via exact `Special:FilePath` filenames.

Key photographs include Oslo Opera House, MUNCH, Vigeland Sculpture Park, Bryggen, Bergenhus Fortress, Fløibanen, Mount Fløyen, Mostraumen/Osterfjord, Nyhavn, Copenhagen canal tours, Amalienborg Palace, Tivoli Gardens, Copenhagen Airport and the Bergen Railway.

See each Wikimedia Commons file page for its author and Creative Commons/public-domain terms.
